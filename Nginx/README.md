# Nginx学习笔记

## 什么是Nginx

Nginx是一个性能突出的服务器/反向代理服务器软件，由于它的内存占用少，启动极快，**高并发能力强**，在互联网项目中广泛应用。Nginx专为性能优化开发，能经受高负载的考验，有报告表明，支持高达50，000个并发连接数。
